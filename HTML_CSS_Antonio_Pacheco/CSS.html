CSS

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags <style>. Também é possível, em vez de colocar a formatação dentro do documento, criar um link para um arquivo CSS que contém os estilos.

A minha opinião sobre CSS parte um bocado da minha opinião sobre HTML.
Sempre vi isto como um bicho de 7 cabeças a espera de me atormentar e acabou por o fazer no meu projeto final do secundário no qual escolhi um template 5* e acabei por desconfigurar aquilo tudo.
(Peço desculpa à pessoa que criou o template, juro que não foi por mal)
Mas estou a gostar imenso de CSS!
Talvez não a um nível em que possa vir a ser Front-End developer mas para desenrascar um site ou para fazer algo personalizado.
Mais uma vez um obrigado ao professor António por fazer me gostar destas tecnologias.

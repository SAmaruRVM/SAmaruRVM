Java

Java é o ambiente computacional, ou plataforma, criada pela empresa estadunidense Sun Microsystems, e vendida para a Oracle depois de alguns anos. A plataforma permite desenvolver programas utilizando a linguagem de programação Java.

Bem, da minha parte o que penso sobre Java? Como já tinha tido experiência em C#, Java tem sido um pouco mais fácil mas é algo que apenas quero despachar.
Sem ofensa à professora Fátima pois nota-se que é uma das suas linguagens preferidas e tem dado a matéria de uma forma muito eficaz e amigável.
Mas parece-me que Java é apenas uma cópia do C# embora C# tenha aparecido anos depois.
Mas não vou mentir, tem sido um benção para relembrar e até aprender novos conceitos relacionados com a POO.
Secalhar futuramente até mudo de ideias sobre a linguagem.

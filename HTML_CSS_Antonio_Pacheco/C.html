C

Cé uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela Organização Internacional para Padronização, criada em 1972 por Dennis Ritchie na empresa AT&T Bell Labs para desenvolvimento do sistema operacional Unix.

Foi a primeira linguagem que dei na ATEC.
Curiosamente foi a primeira linguagem funcional com a qual eu estive contacto. E até agora tenho gostado imenso e até era algo que me via a trabalhar futuramente.
É das poucas linguagens que posso dizer que me tem dado pica de investigar e aprofundar o seu conteúdo. Tanto que decidi investir num livro e até agora tem sido espetacular.
Estou um pouco arrependido de nunca ter pegado em C anteriormente. É uma linguagem que sem dúvida é perfeita para a primeira linguagem de programação ou para quem já tem experiência em programação mas quer trabalhar em linguagens de baixo nível. E isso é o fantástico de C, o facto de podermos manipular diretamente a memória, escolher o que queremos mandar para o compilador ou para o Pré-Processador.

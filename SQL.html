SQL

Structured Query Language, ou Linguagem de Consulta Estruturada ou SQL, é a linguagem de pesquisa declarativa padrão para banco de dados relacional. Muitas das características originais do SQL foram inspiradas na álgebra relacional.

SQL é daquelas linguagens que eu não sabia que ia gostar assim tanto.
Foi algo que dei antes de ir para o meu primeiro estágio (onde eu já sabia que ia depender de SQL para fazer uma aplicação) e nunca aprendi por ninguém.
Tive professores tão bons ao ponto de me passarem pdfs sobre os comandos SQL com um olhar na cara que traduzia para "Aprendes se quiseres, senão quiseres não aprendes".
Tem sido muito bom aprender SQL com a professora Fátima.
Há certos professores que não tem ideia do impacto ou da influência que têm nos alunos (no bom sentido).
